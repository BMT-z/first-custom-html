Basics

The actual command is in the brackets and will be followed a brief simple description

Touch ( touch ) - creates files

- The touch command is used to create files for example ( touch test.txt ) the format of the file needs to be included after the name of the file

Note: using a / will cause the command to not work, doing the command (touch t/est.txt) will cause the error touch: "cannot touch 't/est.txt': No such file or directory"

Ls ( ls ) shows all files

This command lists all the files in whichever directory you are in and is useful when looking for files before you navigate to them, Note: lowercase L
